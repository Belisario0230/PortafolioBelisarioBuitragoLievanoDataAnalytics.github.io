Implementación de Modelos de Datos y Visualización Interactiva en SQL Server y Power BI

El proyecto "Implementación de Modelos de Datos y Visualización Interactiva en SQL Server y Power BI" se centró en diseñar un sistema integral de análisis de datos que solucionara necesidades críticas en áreas como Compras, Producción, Almacén, Ventas, Recursos Humanos y Clientes. A través del uso de SQL Server, se realizaron tareas clave como la restauración de bases de datos, la creación de diagramas Entidad-Relación y consultas avanzadas para relacionar tablas y generar vistas. Posteriormente, los datos fueron transformados en Power Query, optimizando su formato y creando tablas clave como la Tabla Calendario y la Tabla de Actualizaciones.

Con DAX, se enriquecieron los datos mediante cálculos avanzados e inteligencia de tiempo para análisis histórico y proyecciones. Finalmente, se desarrollaron dashboards interactivos en Power BI con indicadores clave específicos para cada proceso, permitiendo un control eficiente, visualizaciones personalizadas y toma de decisiones informada en tiempo real.

Este proyecto refleja la capacidad de integrar herramientas de análisis y visualización para resolver problemáticas empresariales y optimizar la gestión de procesos clave.

Análisis Predictivo de Patrones de Números en los Sorteos de Baloto

El objetivo de este análisis es obtener una visión detallada sobre las características y patrones de los números sorteados en los juegos de Baloto, usando los datos proporcionados. El análisis incluye estadísticas descriptivas, visualizaciones de distribuciones y frecuencias, análisis de correlaciones entre los números, y la identificación de combinaciones recurrentes en los sorteos.
